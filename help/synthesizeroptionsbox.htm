Synthesizer Options Box


The Synthesizer Options Box of the ProSaver Music Manager provides settings that affect the quality of music playback for some audio file types.  Particularly, MIDI files and WAV files can be enhanced by using the DirectX Audio synthesizer provided with DirectX 8.1.  (Because of proprietary encoding issues, MP3 files cannot be enhanced.)  There are two synthesizer options to choose from:

 

• Windows Default – The default synthesizer set up on your system (usually a generic MIDI driver) will be used.  While this is sufficient for MIDI file playback, it is recommended that you use the DirectX Audio synthesizer, because the audio quality is much better.  (If ProSaver does not detect DirectX 8.1 or higher on your system, you will be asked if you would like to download and install it.  This step is highly recommended.)

 

• DirectX Audio – This is the synthesizer of choice for MIDI files and for enhancement of WAV and certain other audio file types.  The DirectX Audio synthesizer allows you to add Audio Effects during playback which make for a richer, fuller sound and a much more enjoyable listening experience.

 

Please note that your synthesizer selection will only be used during screensaver playback.  In the Music Manager, the Windows synthesizer is always used for previewing MIDI files.
