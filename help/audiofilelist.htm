Audio File List


The Audio File List in the ProSaver Music Manager is used to help you navigate your computer’s folders while you decide which audio files to include in your playlists.  The Audio File List has been designed to function in a way that is similar to the customary Windows Explorer-style dialog boxes.  This should make it very easy for those familiar with Windows to use the list, and it is also very easy for newcomers to learn as well.

 

The Audio File List is used in conjunction with a number of additional controls, which make navigation much easier.  You can perform the following operations in the Audio File List:

 

• Double-click a folder, or highlight the folder and press the Enter key, to open the folder.

 

• Double-click a music file, or highlight the file and press Enter, to play the file.

 

• Right-click on a folder and you will be given the option to open the folder.

 

• Right-click on an audio file and you will be given the options of playing the file or adding it to the current playlist.
