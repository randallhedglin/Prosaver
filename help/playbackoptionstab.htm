Playback Options Tab


The Playback Options Tab allows you to set up ProSaver Screensaver Manager according to your own personal preferences.  There are two main sections under the Playback Options tab:

 

• General Settings

 

• Advanced Settings

 

 

For more help in using the Playback Options Tab, see the following ProSaver tutorials:
