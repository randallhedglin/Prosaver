ProSaver Main Window


The ProSaver Main Window is where all the action takes place!  From here, you can create and update screensaver playlists, enter information to help organize your screensavers, set playback options just the way you like them, and much more.  There are several ways to access the ProSaver Main Window:

 

• If hotkeys are enabled (under the Playback Options Tab), you can use the hotkey combination that you have chosen to open ProSaver right away.  (The default hotkey is Ctrl+Alt+P.  Go ahead and try it!)

 

• If the ProSaver icon is displayed in your system tray, just double-click it to open the ProSaver main window.  (Alternatively, you can right-click the icon and select ‘Open ProSaver Options’ from the popup menu.)

 

• Open the Windows Control Panel ‘Display Properties’ screen, either by using the Start menu or by right-clicking on your desktop and selecting ‘Properties’.  From here, go to the ‘Screen Saver’ tab and push the Settings button.
