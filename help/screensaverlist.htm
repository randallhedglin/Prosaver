Screensaver List


The Screensaver List shows all of the screensavers that are available on your system.  There are two display modes for the list that you can choose from:

 

Details Mode (shown above)

 

This mode displays your screensavers, including name, theme, and rating.  Screensavers in the list can be sorted by clicking on the header-bar buttons at the top of the list.  (Click on ‘Name’ to sort by name, ‘Theme’ to sort by theme, or ‘Rating’ to sort by rating.)  Clicking once will sort the list in ascending order (A-Z).  Clicking again will sort in descending order (Z-A).

 

Right-click the list to display a popup menu with the following options:

 

• Add to Playlist – Select this option to add the highlighted screensaver to the current playlist.


• Edit Info – Select this option to open the Screensaver Info Editor for the highlighted screensaver.

 

• Change Rating – This option opens a submenu that lets you easily change the rating for the highlighted screensaver.  Options range from Not Rated, to 1, 2, 3, 4 or 5 Stars.

 

• Show Details – Uncheck this option to switch to List Mode.

 

• Show Rating Images – If this option is checked, ratings are displayed as star images (as shown above).  Otherwise, ratings are displayed as text.

 

List Mode (not shown)

 

This mode displays your screensavers in a simple list, showing only the name of the screensaver.  If you leave the mouse cursor over an item for a brief period, a small popup will appear showing name, theme and rating.  (This option can be enabled or disabled in the Preferences Window.)

 

Right-click the list to display a popup menu with the following options:

 

• Sort by Name – Select this option to sort the screensavers alphabetically.  Select once to sort in ascending order (A-Z).  Select again to sort in descending order (Z-A).

 

• Add to Playlist – Select this option to add the highlighted screensaver to the current playlist.

 

• Edit Info – Select this option to open the Screensaver Info Editor for the highlighted screensaver.

 

• Show Details – Check this option to switch to Details Mode.

 

• Show Popup Info – If this option is checked, the screensaver info popup will be displayed.  Otherwise it will not be.
