Music Manager


The ProSaver Music Manager allows you to create audio playlists for use as background music with your screensavers, and also contains options for playback of audio files.  The Music Manager is divided into five sections:

 

• Audio Files Box

 

• Current Playlist Box

 

• Current Selection Box

 

• Playback Controls Box

 

• Synthesizer Options Box

 

Along the bottom of the Music Manager window are the following buttons:

 

• Help – Click this button to view the ProSaver help file (this file).

 

• Download DirectX 8.1 – NOTE: This button is only visible if ProSaver detects that DirectX 8.1 is not installed on your system.  If this button appears, click on it to be taken to the WaveBreak Software website, where you can download DirectX 8.1.  DirectX provides a number of advanced features for audio playback (especially MIDI files), which can greatly enhance the sound quality of an audio file.  It is strongly recommended that DirectX 8.1 be installed so that you can more fully enjoy this program.

 

• OK – Click this button to save your changes and close the Music Manager window.

 

• Cancel – Click this button to close the Music Manager without saving your changes.

 

• Apply – Click this button to save your changes without closing the window.
