Themes and Ratings


The ProSaver Screensaver Info Editor allows you to assign a theme and a rating to each of your screensavers.  There are two list boxes that allow you to make these selections:

 

• Theme – This box shows the list of themes from which you can choose.  If you would like to assign a theme but do not see it in the list, just click the Edit Themes button to open the Themes Manager.

 

• Rating – This box allows you to select a rating from the following choices:  Not rated, 1 Star, 2 Stars, 3 Stars, 4 Stars, and 5 Stars
