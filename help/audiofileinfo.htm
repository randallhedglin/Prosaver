Audio File Info


The Audio File Info box of the ProSaver Music Manager displays information about the audio file that is currently selected in either the Audio File List or the Audio Playlist.

 

The box displays the song name, file type, and length, as well as any additional information that can be extracted from the file.  (Different types of files include different types of information, some more than others.)
