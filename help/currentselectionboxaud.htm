Current Selection Box


The Current Selection Box of the ProSaver Music Manager displays information about and lets you preview the audio file that is currently highlighted in either the Audio File List or the Audio Playlist.  This section is divided into two parts:

 

• Audio File Info

 

• Audio Controls

 

 

For more information on using the Current Selection Box, see the following ProSaver tutorial:
