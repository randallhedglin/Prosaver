Screensaver Playlist


The Screensaver Playlist displays the list of screensavers that will be shown the next time ProSaver runs in playback mode.  The list will be played back based on your settings under the Playback Options Tab.

 

There are several ways to arrange screensavers in the list:

 

• Use the Move Selection Buttons to arrange playlist items according to your preferences.

 

• Use the ‘Sort Items By’ box to sort screensavers by name, theme, or rating.

 

• You can also use the keyboard to arrange items.  Select an item using the up and down arrow keys.  Move the item by holding the Shift key and using the up and down arrow keys.  Move an item to the top or bottom of the list by holding the Shift+Ctrl keys and pressing either the up or down arrow key.

 

Right-click on the Screensaver Playlist to access a popup menu with the following commands:

 

• Remove Invalid Items – This item will be displayed only if there are invalid (no longer available) screensavers in the playlist.  Select this option to delete all invalid items from the playlist.

 

• Sort Items by – This item opens a submenu that allows you to sort playlist items by name, theme, or rating.

 

• Clear Playlist – Select this item to remove all items from the playlist.

 

• Save to File – Select this item to save the current playlist to a file so that it can be loaded again later.

 

• Load from File – Select this item to load a previously saved playlist from file.

 

• Show Popup Info – If this item is checked, a small popup will appear showing a screensaver’s name, theme, and rating.  Otherwise, no popup will appear.
