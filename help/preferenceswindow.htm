Preferences Window


Click the Preferences button in the Additional Buttons area of the ProSaver Main Window to open the Preferences Window.  The Preferences Window controls how certain features of the Main Window are displayed.  There are several options that can be set:

 

• Enable sound effects – If this option is selected, some small sound effects will be played during interaction with the ProSaver Main Window.

 

• Show tooltips on buttons – Select this option to display a brief reminder of the functions of buttons which contain icons instead of text.

 

• Show splash screen at startup – If this box is checked, a small window is briefly displayed when the ProSaver Main Window is opened and before screensaver playback begins.

 

• Show details in screensaver list – If this option is selected, the Screensaver List will be displayed in Details Mode.  Otherwise, it will be displayed in List Mode.

 

• Show ratings as images – If this option is selected, the Screensaver List will show ratings using star images.  Otherwise, ratings will be shown as text.

 

• Show popup screensaver info – If the Screensaver List is displayed in List Mode, selecting this option will display a small popup containing the screensaver name, theme, and rating if the mouse is left over the screensaver title for a brief period of time.

 

• Show popup playlist info – If this box is checked, both the Screensaver Playlist and Audio Playlist will show small popups containing additional information about the items in the list.
