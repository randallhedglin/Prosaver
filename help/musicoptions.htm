Music Options


The ProSaver Screensaver Info Editor allows you to assign audio files or audio playlists to accompany each of your screensavers.  To select a file, click the Change button to open the Change Music Dialog.  The selected file will be displayed in the ‘Music’ box.

 

There are two playback options available when a valid ProSaver audio playlist has been selected:

 

• Cycle – Songs in the playlist will be played in order

 

• Shuffle – Random songs will be selected from the playlist file.

 

 

For more information on using Music Options, see the following ProSaver tutorial:
