Part A: The System Tray Icon


One of the valuable, time-saving features of ProSaver Screensaver Manager is the System Tray Icon.  This part of the Tutorial will show you how this seemingly insignificant icon can save you lots of valuable time!  Here’s how:

 

1. Look in your system tray.  This is the part of your screen, usually in the lower-right corner, that displays the current time and lots of little icons.  After you install and run ProSaver, one of them will be the ProSaver icon (a yellow circle with the letters ‘P’ and ‘S’ as shown above).  If you do not see the icon, do the following:

 

            a. First of all, look for a little arrow in the system tray.  Clicking this will show some icons that the Windows system has hidden.  (It hides them if it decides that you don’t use them enough.)  If you still don’t see the ProSaver icon, continue on to step (b).

 

            b. Open the ProSaver Main Window, and click on the Playback Options Tab.

 

            c. If necessary, click the Show Advanced Options button to display the Advanced Settings section.

 

            d. Near the middle of the Advanced Settings, you will find three checkboxes.  The top one reads ‘Show ProSaver icon in the system tray’.

 

            e. Make sure this box is checked (click it if necessary) and then click OK to save and close the window.  The icon should now appear.

 

2. Now that you have the icon, double-click on it.  The ProSaver Main Window will open for your own personal screensaver managing enjoyment.

 

3. Right-click on the icon to bring up a popup menu.  The popup menu allows you to do the following things:

 

            a. Run Screensaver Now:  Click this option to start the screensaver immediately.  Useful at times when you are leaving your desk, or when it’s been a long day and you just need a little rest and relaxation, screensaver style.

 

            b. Open ProSaver Options:  This is another easy way to open the ProSaver Main Window, which you will want to do anytime you download a new screensaver or decide it’s time to try out a new playlist for the day.

 

            c. Show Display Properties:  This option provides a convenient way to open the Windows Control Panel to the Display Properties section.  You can use this to configure some aspects of how ProSaver is run, including the time delay before starting the screensaver and adding password protection to the screensaver.

 

            d. Hide ProSaver Icon:  You can use this option if you decide you want to (gasp!) remove the ProSaver icon from the system tray.

 

            e. How to Order:  Displayed in the Trial Version only.  Click this option to be taken to the How to Order window for ProSaver Screensaver Manager.



In addition to the System Tray Icon, ProSaver offers some keyboard shortcuts that can speed things up even further.  These will be discussed as we move on to the next section.
