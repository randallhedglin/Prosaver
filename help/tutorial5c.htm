Part C: Arranging the Playlist


The Current Playlist Box, in the upper-right corner of the ProSaver Music Manager, is where you create and edit audio playlists for use with your screensavers.

 

Here are some things to try with the Audio Playlist:

 

1. First, make sure the playlist you created in Part B of this Tutorial is visible in the list.  If not, use the Load from File button and load the ‘Space Songs.apl’ file back into the playlist.

 

2. Right-click the list to open a popup menu.  Move the mouse pointer to ‘Sort Items by’, and then, in the submenu, click on ‘Name (A-Z)’.  The files will now be arranged in alphabetical order.

 

3. You can do the same thing using the ‘Sort Items by’ box directly underneath the Audio Playlist.  Click on the arrow to drop the list, then click on ‘Name (Z-A)’.  The list will now be sorted in reverse alphabetical order.

 

4. Next, click on an audio file in the playlist.  Using the Move Selection Buttons, practice moving the audio file up, down, to the top, and to the bottom.

 

5. You can also use some keyboard shortcuts to arrange items in the playlist:  Highlight an item using the up and down arrows keys.  If you hold down the Shift key while moving up and down, the selected file will move with you.  You can also hold the Shift+Ctrl keys while using the arrow keys to move the file to the top or bottom of the playlist.

 

6. Use the Delete button to remove one of the files from the list.

 

7. Add the file that you deleted back into the playlist, and spend some time arranging them in just the right order.

 

8. Click the Save to File button and save the file once again as ‘Space Songs.apl’.

 

 

With your playlist arranged just the way you like it and saved to disk, your are now ready to move on to the next section
