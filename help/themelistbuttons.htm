Theme List Buttons


The following buttons are used in conjunction with the Themes List:

 

• New – Click this button to open the New Theme Dialog.  The new theme will be added to the Themes List.

 

• Delete – Click this button to delete the selected them from the list.

 

• Rename – Click this button to open the Rename Theme Dialog.  The theme you enter will replace the currently selected theme.  (Note that if you choose to rename a theme that is currently in use, the ability to cancel the dialog box will be taken away.)

 

• Reset Defaults – Click this button to replace all themes in the list with the default themes list that is supplied with ProSaver Screensaver Manager.

 

• Help – Click this button to view the ProSaver Help file (this file).

 

• OK – Click this button to save all changes and close the dialog box.

 

• Cancel – Click this button to close the dialog box without saving any changes.

 

• Apply – Click this button to save your changes without closing the dialog box.

 

Please note that two default themes (None and Windows) cannot be deleted or renamed.
