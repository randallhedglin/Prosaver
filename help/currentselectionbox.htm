Current Selection Box

 

The Current Selection box displays information about the screensaver that is currently selected in either the Available Screensavers Box or the Current Playlist Box.  It also allows you to change screensaver settings, preview the screensaver, and enter information about the screensaver.  The following topics are covered:

 

• Screensaver Info Box

 

• Screensaver Buttons

 

• Mini-Preview Window
