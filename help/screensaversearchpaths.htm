Screensaver Search Paths


The Screensaver Search Paths section of the Advanced Settings area allows you to specify additional folders for ProSaver to search for screensavers.  Any screensavers found in these additional locations will be added to the Screensaver List.  This feature is useful if you prefer to install screensavers into locations other than the standard Windows folder, or if your system settings do not allow you to modify the Windows folder.  There are three buttons that accompany the Screensaver Search Path list:

 

• Add – Click this button to open a dialog box that will allow you to choose a new folder to add to the list.

 

• Change – Click this button to change the currently highlighted path in the list.  A dialog box will open to allow you to select a replacement folder.

 

• Remove – Click this button to remove the currently selected path from the list.

 

It should be noted that ProSaver will not allow you to set search paths on removable-media devices (floppy drives, CD-ROM drives, etc.)

 

Also, two default folders (the Windows folder and the Windows System folder) cannot be changed or removed from the list.
