Part B: Error Logging and Other Options


The remainder of this Tutorial outlines ProSaver’s Error Logging features, as well as the remaining options that are available under the Advanced Settings.  Are we ready?  Then here we go:

 

1. Make your way once again to the Advanced Settings.

 

2. Take a look at the Error Log box. Here you are given the option to turn error logging on or off, and you can also decide how long the error log file should be.

 

3. Assuming you have run some screensavers by now (and what ProSaver Master hasn't?), you can click the View Error Log button to see its contents.  While this may not look like very interesting reading to the average ProSaver user, it can be very helpful for the WaveBreak Software Technical Support staff if you are having any problems running the program.

 

4. The remaining options are as follows:

 

            a. Show ProSaver icon in the system tray:  Select this option to display the ProSaver System Tray Icon.  The benefits of using this feature were discussed in Tutorial #6: Shortcuts and Time-Savers.

 

            b. Warn if ProSaver is not the current screensaver:  This is a useful little feature that helps to keep ProSaver Screensaver Manager running correctly.  As you may recall, in order to do its job, ProSaver must be set as your current screensaver.  When you install new screensavers, they will sometimes change this setting automatically.  When this box is checked, ProSaver will run a test each time it starts to make sure that it is the current screensaver.  If not, it will give you the option to change it back with minimal effort.

 

            c. Hide advanced options by default:  Check this box to hide the Advanced Settings when ProSaver starts.

 

            d. Restore Blocked Messages:  As you use ProSaver, you will notice that some message boxes give you the option to not display the message again.  If you click the Restore Blocked Messages button here, all of the messages you have decided not to show will be shown once again.  (Of course, you can always decide not to show them again later.)

 

 

And that’s it!  You have completed Tutorial #7: Using Advanced Options.  If you have finished all of the ProSaver Tutorials, then you can tell all of your friends that you are a ProSaver Grand Master.  They will probably laugh at you and talk about you when you’re not listening, but, hey, people like that just can’t understand, can they?  So please use your newfound knowledge wisely, and enjoy using ProSaver Screensaver Manager!
