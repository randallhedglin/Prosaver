System Tray Icon


The ProSaver Screensaver Manager system tray icon, if enabled, can usually be found in the bottom-right portion of your computer screen.  This icon serves a number of useful purposes:

 

• To quickly open the ProSaver Main Window, just double-click the icon.

 

• Right-clicking the icon opens a popup menu with some useful options, including ‘Run Screensaver Now’, ‘Open ProSaver Options’, ‘Show Display Properties’, and, if you are using the Trial Version, ‘How to Order’.

 

Additionally, the background application that runs the icon also takes care of the hotkeys that allow you to conveniently open the ProSaver Main Window or start your screensaver with the push of a button.

 

To enable the ProSaver system tray icon:

 

If the system tray icon is not visible, perform the following steps to make it visible:

 

• Open the ProSaver Main Window.

 

• Click on the Playback Options Tab.

 

• If necessary, click the Show Advanced Options button to display the Advanced Settings.

 

• Check the ‘Show ProSaver icon in the system tray’ box.  When you push OK or Apply, the icon will become visible.
