Playback Controls Box


The Playback Controls Box of the ProSaver Music Manager provides settings for hotkeys that function during screensaver playback.  There are four controls available:

 

• Next – Use this key combination to skip to the next song in the audio playlist during screensaver playback.

 

• Vol+ – Use this combination to increase the volume during screensaver playback.

 

• Vol- – Use this combination to decrease the volume during screensaver playback.

 

• Mute – Use this hotkey to mute the volume during screensaver playback.

 

It is important to note that these keys will not work in the Music Manager!  They are provided only during screensaver playback.  To enable these hotkeys, check the ‘Enable controls’ box.  To disable them, clear the checkbox.

 

Another important note:  During screensaver playback, these controls will not function immediately – it takes several seconds for the audio and playback control system to initialize.  Once the music starts, it is safe to use the hotkeys.  (Trying to use the hotkeys before music playback has started will simply close the screensaver.)
