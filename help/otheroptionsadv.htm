Other Options


There are three additional options and two additional buttons in the Advanced Settings area of the Playback Options Tab.

 

The additional options are:

 

• Show ProSaver icon in the system tray – If this option is selected, a small ProSaver icon is displayed in the system tray area of your screen.

 

• Warn if ProSaver is not the current screensaver – If this box is checked, ProSaver will check your current screensaver at startup.  If ProSaver is not your current screensaver, a message box will appear asking you if you would like to change this setting.  (ProSaver must be set as your current screensaver in order to function correctly.  See the article ProSaver and Windows Control Panel for more information.)

 

• Hide advanced options by default – If this option is selected, the Advanced Settings area will be hidden when the Playback Options Tab is first displayed.

 

The additional buttons are:

 

• Restore Blocked Messages – Some message boxes in ProSaver Screensaver Manager have a small checkbox which reads ‘Do not show this message again’.  Checking the box causes the message to not be displayed again.  Clicking this button will reset all of these message boxes so that they will all be displayed.

 

• Hide Advanced Options – Click this button to hide the Advanced Settings area.
