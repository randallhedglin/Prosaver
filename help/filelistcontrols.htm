File List Controls


The are a number of helper controls that are used in conjunction with the Audio File List in the ProSaver Music Manager.  They are as follows:

 

• Look-in list – Click the arrow and this list will display a tree-like representation of the path to the current folder, all the way back to the root folder (usually the Desktop).  This provides an easy way to change folders or disk drives.

 

• Folder-back button – Click this button to return to the previously displayed folder.

 

• Folder-up button – Click this button to move to the parent folder of the currently displayed folder.

 

• Files-of-type list – Click the arrow to see a list of file types that can be filtered for display in the Audio File List.  (In other words, only the selected file types will be visible.)

 

• Clear Selection button – Click this button to clear the current selection.  The Audio File Info box will be reset, and the audio file (if it is playing) will be stopped.

 

• Add to Playlist button – Click this button to add the highlighted music file to the current playlist.
