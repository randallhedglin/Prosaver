Screensaver Playlist Buttons


There are three additional buttons associated with the Screensaver Playlist:

 

• Clear List – Click to remove all items from the current playlist.

 

• Save to File – Click to save the current playlist to file so that it can be loaded again later.

 

• Load from File – Click to load a previously saved playlist from file.
