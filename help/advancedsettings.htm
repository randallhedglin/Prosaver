Advanced Settings


The Advanced Settings section of the Playback Options Tab contains a number of options that are intended for advanced computer users.  (Everyday users of ProSaver Screensaver Manager will probably never need to use these options, but it is good to know that they are available.)

 

NOTE: By default, the Advanced Settings are hidden from the Playback Options Tab.  If you would like to view these options, click the Show Advanced Options button.

 

The Advanced Settings section is divided into the following areas:

 

• Screensaver Search Paths

 

• Error Logging

 

• Other Options
