ProSaver and Windows Control Panel


  The purpose of this section is to describe the interaction between ProSaver Screensaver Manager and Windows Control Panel.

 

  The Windows Control Panel is the part of the Windows operating system that runs and manages the screensaver.  Normally, only one screensaver can be selected.  ProSaver Screensaver Manager works by extending this part of Windows to allow a selection of screensavers to run, and even lets you add background music to them.  Here’s how:

 

  • You select a certain number of minutes for the computer to wait before running the screensaver.  This wait time is selected under the Windows Control Panel ‘Display Properties’ box.

 

  • If you let the computer sit (with no mouse or keyboard input) for this length of time, the screensaver starts.  In this case, Windows opens the ProSaver Playback Window.  This window is the one that runs all of the screensavers in your playlist.

 

  • When you close a screensaver in the normal way (by moving the mouse or pressing a key), the screensaver closes, and so does the ProSaver Playback Window.  This lets you get right back to what you were doing before.  (If, for some reason, the Playback Window fails to close, just click on it and it will go away.)

 

  Now, here’s the important part:  For ProSaver to work correctly, it must be set as your current screensaver in the ‘Display Properties’ box.  This way, Windows will start the ProSaver Playback Window when your time has elapsed.  Windows also allows you to password-protect the computer when the screensaver exits, which you can also do from the ‘Display Properties’ box.  (However, for security reasons, Windows XP will not allow the audio hotkeys to function if the password-protect box is checked.)
