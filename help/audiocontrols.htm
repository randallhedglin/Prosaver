Audio Controls


A number of Audio Controls are available for previewing songs in the ProSaver Music Manager.  (These controls are available only when a file is selected in either the Audio File List or the Audio Playlist.)  Controls are as follows:

 

• Play button – Click to begin playing the file at its current position, or to resume after clicking the Pause button.

 

• Pause button – Click to temporarily pause the file at its current position.

 

• Stop button – Click to stop playback and set the file position back to the beginning.

 

• Full Rewind button – Click to set the file position back to the beginning.  (If the file is playing, it will resume playing from the beginning.)

 

• Rewind button – Click to move the file position back a small increment.

 

• Fast Forward button – Click to move the file position forward a small increment.

 

• Position Slider control – This control displays the current position as well as the full length of the audio file.  Drag the slider left or right to change the current position of the file.
