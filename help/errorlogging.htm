Error Logging


The Error Logging section of the Advanced Settings area contains options for the logging of errors during screensaver playback.  Error logging is useful in situations where ProSaver playback for a particular screensaver or audio file is not functioning properly.  (ProSaver will not display any error messages on the screen so as not to interrupt playback.)  If the box is checked, all errors will be recorded to an error log file of the size specified in the ‘Max lines’ box.  Click the View Error Log button to view the current contents of the error log.
