General Settings


The General Settings section of the Playback Options Tab allows you to run the ProSaver Screensaver Manager Playback Window according to your own preferences.  The General Settings section is divided into the following areas:

 

• Screensaver Options

 

• Playback Order Options

 

• Screensaver Hotkeys

 

• Other Options

 

 

For more help in using General Settings, see the following ProSaver tutorial:
