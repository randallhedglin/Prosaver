Move Selection Buttons


The Move Selection Buttons allow you to arrange audio files in the Audio Playlist.  There are five buttons:

 

• Move Up – Click to move the selected file up one position in the list.

 

• Move Down – Click to move the selected file down one position in the list.

 

• Move to Top – Click to move the selected file to the top of the list.

 

• Move to Bottom – Click to move the selected file to the bottom of the list.

 

• Delete – Click to remove the selected file from the playlist.
