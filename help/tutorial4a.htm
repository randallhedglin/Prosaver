Part A: The Screensaver Info Editor


The main tool you will use in organizing your screensavers is the Screensaver Info Editor.  With it, you can do the following things with each of your screensavers:

 

• Assign a theme (or, category) to a screensaver.  ProSaver provides a useful list of default themes, but if you do not see the one you want, it’s very easy to add it to the list!

 

• Give the screensaver a rating, from one to five stars.  This way, you can arrange the Screensaver List to show the best ones first.

 

• Add music or a music playlist to the screensaver.  (This topic will be discussed in more detail in Tutorial #5: Adding Music and Audio Playlists.)

 

• Provide a useful description for the screensaver to help you find it again later.  (Default descriptions are provided for each of the default Windows XP screensavers.)

 

• Turn off the mini-preview for a screensaver in a situation where the mini-preview does not work correctly or causes other problems.

 

 

Let’s get started:

 

1. Open the ProSaver Main Window.

 

2. Look in the Screensaver List for one of your favorite screensavers.  Click on it.

 

3. You will now see some information about the screensaver in the Screensaver Info Box.  Also, the Mini-Preview Window will begin showing a small preview of the screensaver.

 

4. In the Current Selection Box, click the Edit Info button to open the Screensaver Info Editor.

 

5. And that’s all there is to it!  While we’re here, let’s give this screensaver a theme.

 

6. Click the arrow in the ‘Theme’ box to open the list of themes, and choose one for this screensaver.

 

7. Click OK to close the Screensaver Info Editor.  You will notice that the theme you selected from the list will appear in the Screensaver Info Box, along with the current screensaver.

 

 

But wait!  What if you want to use a theme that’s not in the list?  To find out what to do, let’s move on to the next section.
