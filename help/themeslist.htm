Themes List


The Themes List of the Themes Manager displays all of the themes that are available to be assigned to screensavers.  The Themes Manager allows you to add or remove themes, based on your preferences, using this list.

 

In addition to the buttons in the Themes Manager window, you may choose to right-click on the Themes List to bring up a popup menu with the following options:

 

• Rename – Rename the selected theme using the Rename Theme Dialog.  Please note that if you choose to rename a theme that is currently in use, the ability to cancel the dialog box will be taken away.

 

• Delete – Delete the selected theme.

 

 

For more information on using the Themes List, see the following ProSaver tutorial:
