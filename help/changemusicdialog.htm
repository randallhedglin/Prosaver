Change Music Dialog


The Change Music Dialog is displayed by the Screensaver Info Editor whenever you click the Change button.  This dialog box allows you to perform several operations:

 

• Load Single Audio File – Click this button to choose and assign an audio file (such as MP3, WAV or MIDI) to play as background music with the selected screensaver.

 

• Load ProSaver Audio Playlist – Click this button to select a ProSaver audio playlist to go along with the selected screensaver.

 

• Set Audio to None – Click this button to turn off music for the selected screensaver.

 

• Open Music Manager – Click this button to open the Music Manager, which allows you to create and modify ProSaver audio playlists and set audio options according to your preferences.  (When you close the Music Manager, you will be given the option of assigning the last-saved playlist to the current screensaver.)

 

• Help – Click this button to view the ProSaver help file (this file).

 

• Cancel – Click this button to close the Change Music Dialog without making any changes.
