Screensaver Description


The Screensaver Description area of the ProSaver Screensaver Info Editor allows you to enter a description (up to 1000 characters) for the selected screensaver.  The number of characters remaining is displayed above the text editor.  A description has been provided for each of the Windows XP default screensavers, which you can edit if you prefer.
