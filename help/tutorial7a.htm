Part A: Screensaver Search Paths


Some users may decide to install screensavers into locations other than the default Windows folders.  There are a few reasons for this:

 

• It can make the screensaver files themselves easier to find when they are all stored in a common location.

 

• You may wish to install several (perhaps older) screensavers at one time, and copying them to a single folder is a logical and quick way of doing this.

 

• Some Windows computers are configured so that you cannot add files to the Windows folders.  In this case, the only way to run new screensavers is to install them to another location and run them from there.

 

 

In all of these situations, and no doubt there are others, ProSaver’s Screensaver Search Paths box provides the answer.  Using the list is very simple:

 

1. Make your way to the Advanced Settings section under the Playback Options Tab.  (Since you have proven your worth as a ProSaver Master by making it all the way to Tutorial #7, we won’t bore you with the step-by-step details.)

 

2. Here you will find the Screensaver Search Paths box.  To add a path to the list, just click the Add button.  This will open a standard Windows ‘Browse for Folder’ dialog box, which you can use to select the appropriate path.

 

3. Repeat as many times as necessary to make your life complete.
