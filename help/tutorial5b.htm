Part B: Using the Audio File List


Here’s where things start to get exciting!  In this and the next section, you will learn how to create audio playlists, which can then be used as background music for your screensavers (for cases where you find more than one song that sounds good with a particular screensaver).  Follow these steps to open the Music Manager and get things started:

 

1. Open the ProSaver Main Window.

 

2. Click on the Playback Options Tab, near the top of the window.

 

3. In the lower-right corner of the General Settings section, click on the Open Music Manager button.

 

4. When the Music Manager opens, you will be able to find the Audio Files Box in the upper-left corner of the window.

 

5. The Audio File List has been designed to function as an Explorer-type window, so it’s layout should be familiar to most users of Windows.  For the next step, you will want to have the ProSaver sample audio folder displayed.  (If it isn’t, you can usually find it at ‘C:\Program Files\ProSaver\Audio’.)

 

6. Here’s how to use the various controls that accompany the Audio File List:

 

            a. Click the arrow in the ‘Look in’ box to display a tree-like listing of the current folder and all of its parent folders.  You will also find a list of available hard drives, your ‘Desktop’ folder, and your ‘My Documents’ folder so that you can easily choose different locations.

 

            b. Click the Folder Up button (in the upper-right corner of the window) to move to the parent folder of the current folder.  Keep clicking until you reach the ‘Desktop’ folder.

 

            c. Now, click the Folder Back button (just to the left of the Folder Up button) and click it until you have returned to the sample audio folder.

 

            d. In the Audio File List itself, double-click any of the sample audio files and it will begin to play.  (Click the Stop button in the Current Selection Box to stop the song.)

 

            e. Click the arrow in the ‘Files of type’ box to list the different file types that are available to be selected.  Click the ‘MP3 Files’ selection and the sample audio files will disappear!  (That’s because they are all MIDI files, not MP3 files.)  To make them come back, click the arrow again and choose ‘All Audio Files’.

 

7. And now, back to making a playlist:  Click on the ‘Space Battle.mid’ file, then click the Add to Playlist button.

 

8. The file is in the list!  Now, repeat this process for the following files: ‘Epic Battle.mid’, ‘Alien World.mid’, and ‘Lost and Alone.mid’.

 

9. You should now have four files displayed in the Current Playlist Box.  Click the Save to File button.

 

10. Type ‘Space Songs’ into the file name box and click Save.

 

 

You have just created your first audio playlist!  Next, you will learn how to arrange the playlist.  When you are ready, we will move on to the next section.
