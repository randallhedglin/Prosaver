Turn Off Preview Box



In the ProSaver Screensaver Info Editor, you can choose to disable the Mini-Preview Window for a certain screensaver.  This feature is useful in some situations where a screensaver’s mini-preview causes popup windows or other problems, which can be very annoying.  If any of these things happen, try clicking this box to turn off the mini-preview.
