Part D: Playback Controls and Options


The ProSaver Music Manager also allows you set some options to be used during screensaver playback.  Here’s how to use them:

 

1. Open the Music Manager, as described in Part B of this Tutorial.

 

2. In the bottom-right of the window, you will see two sections, the Playback Controls Box and the Synthesizer Options Box.  First, let’s discuss the Playback Controls Box.

 

3. This box contains four controls that can be used during screensaver playback.  Here is what they do:

 

            a. Next: Skip to the next song in the audio playlist for the screensaver.

 

            b. Vol+: Increase the volume during screensaver playback.

 

            c. Vol-: Decrease the volume during screensaver playback.

 

            d. Mute: Mute the volume during screensaver playback.

 

4. It is important to note that these controls only work during screensaver playback! In other words, you can’t use them while you are in the Music Manager.  They will not have any effect.  The other thing to keep in mind is that you must wait a few seconds after the screensaver begins before using the playback controls.  If not, the screensaver may turn itself off!  When the music has started, it is safe to use the controls.

 

5. You are free to change the controls to anything you would like by clicking inside the box for each one and entering a new combination of the Control, Alt or Shift keys along with one other key.

 

6. Now let’s move on to the Synthesizer Options Box.  This box gives you some control over how synthesized music (mostly MIDI files) are played back.

 

7. This box contains four options for synthesized music playback:

 

            a. Windows Default:  Use the generic MIDI synthesizer that Windows uses.  Functional but boring.

 

            b. DirectX Audio:  Use the DirectX Audio synthesizer.  This is an advanced playback system that provides much better sound quality, and gives you the option of adding some effects to the synthesized output.  (This is the recommended option, and the default setting for this box.)

 

            c. Reverb:  Adds reverberation (an echo-like quality) to the music, which gives the audio a sound like that of a concert hall.

 

            d. Chorus:  Adds a fuller, richer sound to the music in the audio file.

 

8. When starting up, the Music Manager will check your system for the required version of DirectX.  If it is not found, a button will be displayed along the bottom of the Music Manager window, which you can click at any time to download the required files.  It is recommended that you do this, if possible, and use the DirectX Audio synthesizer.  The difference is simply amazing!

 


Whenever you’re ready, let’s move on to the next section, where we will put it all together.
