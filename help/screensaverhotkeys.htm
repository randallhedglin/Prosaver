Screensaver Hotkeys


The Screensaver Hotkeys section of the General Settings area is where you decide if and which hotkeys will be used to activate certain ProSaver Screensaver Manager features.  If the box is checked, hotkeys are enabled and will be defined as shown in the boxes.  Otherwise, hotkeys will not be used.

 

There are three hotkeys that can be set:

 

• Start screensaver – When you push this key combination, your screensaver will start immediately.  This can be useful if you are leaving your desk and want to hide your desktop, or if you just need a little break and want to sit back and relax!  (The default value for this hotkey is Ctrl+Alt+S.)

 

• Skip to next screensaver – If you push this key combination during screensaver playback, the current screensaver will end and the next one from your playlist will be shown.  This is a useful feature for showing off your screensaver collection to friends and colleagues.  (The default value for this hotkey is Ctrl+Alt+N.)

 

• Open ProSaver options – Push this key combination at any time to open the ProSaver Main Window.  If you have just downloaded a new screensaver, or want to modify your playlist for any reason, this will save you a few clicks of the mouse as well as a little valuable time!  (The default value for this hotkey is Ctrl+Alt+P.)
