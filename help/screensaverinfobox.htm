Screensaver Info Box


The Screensaver Info Box displays the following information about the highlighted screensaver:

 

• Icon – The screensaver’s program icon is displayed in the upper-left corner of the box.

 

• Name – The name of the screensaver.

 

• Theme – The theme that you have assigned to the screensaver in the Screensaver Info Editor.

 

• Rating – The rating that you have assigned to the screensaver.

 

• Description – The description that you have given to the screensaver.  (The default Windows XP screensavers are automatically given a description.  You can change the description at any time.)
