Screensaver Buttons


There are three buttons that you can use to perform operations on the selected screensaver:

 

• Settings – Click this button to open the standard ‘Settings’ dialog box for the screensaver.  Each screensaver has its own options that can be set in this way.

 

• Preview – Click this button to run a full-screen preview of the selected screensaver, including any background music that you have selected.

 

• Edit Info – Click this button to open the Screensaver Info Editor for the current screensaver.
