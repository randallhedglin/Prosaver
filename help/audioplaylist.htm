Audio Playlist


The Audio Playlist displays a list of audio files that can be saved and then used as background music for any of your screensavers.  The list will be played back based on the settings you choose in the Screensaver Info Editor.

 

There are several ways to arrange audio files in the list:

 

• Use the Move Selection Buttons to arrange playlist items according to your preferences.

 

• Use the ‘Sort Items By’ box to sort audio files by name or type.

 

• You can also use the keyboard to arrange items.  Select an item using the up and down arrow keys.  Move the item by holding the Shift key and using the up and down arrow keys.  Move an item to the top or bottom of the list by holding the Shift+Ctrl keys and pressing either the up or down arrow key.

 

Right-click on the Audio Playlist to access a popup menu with the following commands:

 

• Remove Invalid Items – This item will be displayed only if there are invalid (no longer available) audio files in the playlist.  Select this option to delete all invalid items from the playlist.

 

• Sort Items by – This item opens a submenu that allows you to sort playlist items by name or file type.

 

• Clear Playlist – Select this item to remove all items from the playlist.

 

• Save to File – Select this item to save the current playlist to a file so that it can be used as screensaver background music.

 

• Load from File – Select this item to load a previously saved playlist from file.

 

• Show Popup Info – If this item is checked, a small popup will appear showing the song name and file type.  Otherwise, no popup will appear.
