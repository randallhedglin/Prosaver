I. What is ProSaver Screensaver Manager?

 

ProSaver Screensaver Manager is a full-featured screensaver management and enhancement application that lets users organize, arrange, and play back all of their favorite screensavers, and even add background music.  All of this and more is accomplished by means of a professional, easy-to-use Windows interface.

 

There are literally thousands of screensavers today that can be downloaded with just a few clicks of the mouse.  They provide all of us with a much-needed break from our hectic routine.  How many times have we returned from the fridge or the water cooler only to be pleasantly surprised by the screensaver running on our desktop?  It can do wonders for our state of mind to just sit back, relax, and watch for a few minutes.

 

Do you have a favorite screensaver?  Most of us do; in fact, the vast majority of computer users, whether at home or in the office, have several favorites that they would like to use.  Unfortunately, Microsoft Windows only allows the selection of a single screensaver at any given time.  ProSaver Screensaver Manager acts as an extension of the Windows Control Panel and overcomes this limitation by allowing the selection of any number of screensavers, which are then played back, one after another, according to the user's preferences.  Whether you like to watch gently flowing waterfalls, morphing geometrics, blasting enemy spaceships in the outer reaches of space, or all of the above, ProSaver Screensaver Manager ensures you will never have to stop using any of you favorite screensavers again!

 

This program has been designed for all users of Microsoft Windows who enjoy using and watching screensavers.  Screensavers have come a long way since the original "blank screen", and today they can truly be described as a form of art.  ProSaver Screensaver Manager is your own personal "art gallery", and you are the one who decides what is included in your collection of masterpieces.

 

For further resources, see our Soapbox article "A brief history of screensavers" at http://www.wavebreaksoftware.com/ssaverhistory.htm.

 

II. Getting started

 

System requirements:

 

OS:                 Windows 98/Me/NT4.0/2000/XP or higher

CPU:              Intel Pentium Pro (or equivalent) 500 MHz or higher

Memory:       32MB RAM

Video:                        Minimum 800x600 display resolution

Hard Disk:    2MB of hard disk space

 

It is also recommended that Microsoft DirectX 8.1 or higher be installed due to the audio enhancement capabilities it provides, particularly for playback of MIDI files.  The most recent version of DirectX is freely available for download from the Microsoft website.  Visit http://www.wavebreaksoftware.com/directx.htm for instructions and help on finding and installing the software.

 

Installation of this software is simple and straightforward.  If the software has been downloaded as a compressed ZIP file, use a program such as WinZip to open/extract the files inside and run the SETUP.EXE program to begin installation.  If the software is contained in a self-extracting EXE file, run it to automatically extract the necessary files to a temporary folder and begin installation.  In either case, when the Setup program begins, just follow the step-by-step instructions on the screen to complete the installation.

 

 

Figure 1: The ProSaver Setup Screen

 

Program removal is also simple and straightforward.  The Setup program creates a program folder in the Start Menu for ProSaver Screensaver Manager, which contains an option to "Uninstall ProSaver".  Click this option to remove the program from your system.

 

III. Program features

 

Creating screensaver playlists:  ProSaver Screensaver Manager allows users to quickly and easily create playlists from the available screensavers on their system.  A concise Screensaver List displays each screensaver's name as well as a user-assigned theme and rating.  A click of the mouse adds a screensaver to the current playlist.

 

 

 

Figure 2: The ‘Screensavers’ Tab

 

The Current Playlist displays a list of the screensavers that will be shown when playback begins.  Items in the playlist can be arranged in any order, and different playlists (based on theme, holiday seasons, etc.) can be saved and loaded as needed.  Newly downloaded screensavers can be added to the list with minimal effort, and "stale" screensavers can be removed just as easily.

 

Setting playback options:  ProSaver Screensaver Manager provides a number of settings for users to play back screensavers according to their own preferences.  Choose to show only one screensaver per playback session or display screensavers from the current playlist as a slideshow, one after another, each one for a specified number of minutes.  Screensavers can also be shown in order from the playlist, or ProSaver can surprise you by selecting random screensavers to display.

 

 

 

Figure 3: The ‘Playback Options’ Tab

 

Advanced options include the ability to add new paths for storing screensavers (other than the default Windows and Windows System folders).  This feature can be useful for users who would like to store screensaver files in a common location, for installing a number of (perhaps older) screensavers at one time, or for installing screensavers on certain systems that do not allow the Windows folders to be modified.

 

Organizing a screensaver collection:  Some users accumulate screensaver collections of massive proportions.  ProSaver Screensaver Manager makes organizing such a collection easy by giving users the ability to assign themes, ratings, and descriptions to each screensaver.

 

 

 

Figure 4: The Screensaver Info Editor

 

ProSaver comes pre-loaded with a number of useful themes, or categories, that can be used for organizing screensavers.  However, there may be some screensavers that do not fit into any of these categories, or users may prefer different ones.  New themes can be added quickly and easily with the  Themes Manager.

 

 

 

Figure 5: The Themes Manager

 

The Screensaver Info Editor also allows users to assign ratings to their screensavers, allowing the best ones to be found quickly.  Additionally, a brief description for each screensaver can be added to remind users of what they like (or dislike) about each screensaver.

 

Shortcuts and time-savers:  ProSaver Screensaver Manager offers users a number of time-saving features when interacting with the program.  A ProSaver icon can optionally be displayed in the Windows system tray area.  Right-clicking this icon opens a menu which includes options to immediately start the screensaver, to open the ProSaver Main Window, and to open the Windows Display Properties box.  Double-clicking the icon also opens the ProSaver Main Window.

 

Even easier than using the icon, a number of hotkeys are available for some common and useful operations.  For example, if you are about to leave your computer, use the 'Start screensaver' key combination to instantly hide your desktop.  If you have just installed a brand new screensaver, use the 'Open ProSaver options' combination to view the screensaver's settings and information and to add it to the current playlist.

 

Additional hotkey operations during screensaver playback include the ability to skip to the next screensaver in the playlist (useful for showing off your screensaver collection to friends or colleagues) and to adjust the volume during playback.

 

Adding background music:  One of the most useful features of ProSaver Screensaver Manager, at least from an artistic point-of-view, is the ability to add background music to most screensavers.  Have you ever thought that a certain screensaver would be just perfect … if only it had a certain song playing in the background?  With ProSaver Screensaver Manager, users can easily adjust audio settings and create music playlists to use as background music for their favorite screensavers.

 

 

Figure 6: The Music Manager

 

Numerous audio formats are supported, including MP3, WAV, MIDI, WMA, AU, and others.  A small number of sample MIDI files is included with the program for users who are new to digital audio.  ProSaver can even be used in many cases to replace the background music that accompanies some screensavers.  In any case, the right music can quickly turn a good screensaver into a great screensaver!

 

ProSaver also takes advantage of the powerful DirectX Audio synthesizer, which is a highly advanced playback system for MIDI and WAV files.  The DirectX synthesizer provides a level of audio quality that is vastly superior to the generic MIDI driver that comes with Microsoft Windows.  ProSaver also takes advantage of some of the audio effects that DirectX can add, including reverberation and chorus effects.  After listening to a few MIDI files in this way, most users agree that DirectX Audio is the best way to listen to their MIDI file collection.

 

A list of useful audio resources can be found on the WaveBreak Software website at http://www.wavebreaksoftware.com/audioresources.htm.

 

Extensive help system:  ProSaver Screensaver Manager includes an extensive Help file that explains every aspect of interacting with the program.  Context help is available for nearly every control, which can be accessed with a simple right-click.  Additionally, the Help file contains a number of tutorials to help new users get up and running quickly, and to help all users take full advantage of the program's features.
